## 🛠️ Useful Aliases

Add to your `~/.zshrc`:

```bash
alias dev='cd ~/Developer'
alias insight='cd ~/Developer/projects/insight'
alias projects='cd ~/Developer/projects'
alias devtools='cd ~/Developer/tools'
```

## 📝 Naming Conventions

- **Projects**: `lowercase-with-hyphens`
- **No spaces** in directory names (use hyphens or underscores)
- **Descriptive names**: Clear purpose from name alone
