# Node.js API Template

Express-based REST API starter.

## Quick Start

```bash
npm install
npm run dev
```

## Stack

- Express
- TypeScript
- Prisma (optional)
- Jest
- ESLint

## Structure

```
src/
├── routes/
├── controllers/
├── middleware/
├── models/
├── utils/
└── server.ts
```

## Scripts

- `npm run dev` - Start with nodemon
- `npm start` - Production start
- `npm test` - Run tests
- `npm run build` - Build TypeScript
