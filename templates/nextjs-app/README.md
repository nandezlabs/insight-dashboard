# Next.js App Template

Full-stack Next.js application starter.

## Quick Start

```bash
npm install
npm run dev
```

## Stack

- Next.js 14+
- React 18
- TypeScript
- Tailwind CSS
- App Router

## Structure

```
app/
├── (routes)/
├── api/
├── components/
└── layout.tsx

public/
└── assets/
```

## Scripts

- `npm run dev` - Development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint
