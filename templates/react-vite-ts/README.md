# React + Vite + TypeScript Template

Basic starter template for React applications.

## Quick Start

```bash
npm install
npm run dev
```

## Stack

- React 18
- Vite
- TypeScript
- ESLint
- Prettier

## Structure

```
src/
├── components/
├── hooks/
├── utils/
├── App.tsx
└── main.tsx
```

## Scripts

- `npm run dev` - Start dev server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
